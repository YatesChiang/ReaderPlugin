### ReaderPlugin
txt文本阅读器插件gradle版本  
旧版本见 [devkit分支](https://github.com/iamyours/ReaderPlugin/tree/devkit)  
如何开发插件见 [官方文档](https://jetbrains.org/intellij/sdk/docs/intro/welcome.html)  

#### 功能
- [x] **LineMarker标记运行方法**
- [x] **通过代码运行生成阅读器**
- [x] **通过添加`Run Configuration`运行生成阅读器**
- [x] **通过编辑`Run Configuration`修改方法阅读参数**
